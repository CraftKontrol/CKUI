## CKUI Support Assistant

This repository provides an AI-powered support assistant specifically designed to help with CKUI troubleshooting and configuration. The assistant is trained on the complete CKUI documentation and can provide instant help with:

### Assistant Capabilities

- **Module Configuration**: Get help setting up CKUI modules including Audio, Content Manager, Sensors, and more
- **Troubleshooting**: Resolve issues with Kinect Azure, Leap Motion, Lidar sensors, and other hardware integrations
- **Project Management**: Assistance with CKUI Project Manager setup and configuration
- **System Integration**: Help with rendering pipelines, stereo engines, and output configurations

### How to Use

1. Open the CKUI Support Assistant web interface : [CKUI Support Assistant](https://craftkontrol.github.io/CKUI/)
2. Enter your Mistral AI API key (get one from [Mistral AI Console](https://console.mistral.ai/))
3. Ask questions about any CKUI module or troubleshooting issue
4. Receive detailed, step-by-step guidance with relevant wiki links

The assistant provides accurate answers based exclusively on the official CKUI documentation and includes direct links to relevant wiki sections for further reading.


## CKUI Project Manager

The CKUI Project Manager is a powerful tool for organizing and managing your CKUI projects. It provides a centralized interface for:

- **Project Creation**: Set up new CKUI projects with predefined templates and configurations
- **Asset Management**: Organize media files, textures, and resources across your projects
- **Module Integration**: Easily add and configure CKUI modules within your project structure
- **Version Control**: Track changes and maintain project versions for better workflow management
- **Export/Import**: Share project configurations and settings between different CKUI installations

For detailed setup instructions and advanced features, refer to the WIKI section in the CKUI [documentation](https://github.com/CraftKontrol/CKUI/wiki/).
